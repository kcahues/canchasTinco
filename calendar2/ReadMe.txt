============ Instructions ============
1. In the dbConfig.php file, specify the database host (DB_HOST), username (DB_USERNAME), password (DB_PASSWORD), and name (DB_NAME) as per your database credentials.

2. Open the index.html file on the browser ===> A full-sized calendar will appear ===> The events will be fetched from the database and listed in date cells.
===> Click on date cell to add event.
==> Click on an event title to view and delete the event.
