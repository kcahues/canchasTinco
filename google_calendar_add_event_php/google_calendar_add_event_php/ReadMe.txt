============ Instructions ============
1. Create a database (for example, codexworld) and import the "SQL/events.sql" file into this database.

2. Open the "config.php" file in a editor: 
===> Specify the database host (DB_HOST), username (DB_USERNAME), password (DB_PASSWORD), and name (DB_NAME) as per your MySQL database credentials.
===> Specify the Client ID (GOOGLE_CLIENT_ID), Client Secret (GOOGLE_CLIENT_SECRET), and Redirect URL (REDIRECT_URI) as per your Google API Console Project.

3. Open the index.php file on the browser and test the event creation and Google calendar event add functionality.
